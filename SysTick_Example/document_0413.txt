    먼저 이전에 커밋된 소스 코드의 커밋 해시(또는 브랜치 또는 태그)를 확인합니다. 이를 위해 다음 명령어를 사용할 수 있습니다:

    git log
    위 명령어를 실행하면 커밋 히스토리가 표시됩니다. 원하는 커밋의 해시를 확인하세요.


    원하는 이전 상태로 돌아가려면 다음 명령어를 사용합니다. 여기서 <commit-hash>는 이전에 확인한 커밋의 해시입니다.

    git checkout <commit-hash>
    예를 들어, 만약 이전에 확인한 커밋의 해시가 abcdef123456라면 다음과 같이 명령어를 실행합니다:

    git checkout abcdef123456
    이렇게 하면 해당 커밋의 상태로 이동됩니다. 필요한 작업을 수행하고 나서, 다시 최신 상태로 돌아가려면 git checkout 명령어를 사용하여 브랜치 이름을 입력하면 됩니다.


    git checkout <branch-name>
    예를 들어, 만약 메인 브랜치가 main이라면 다음과 같이 명령어를 실행합니다:


    git checkout main
